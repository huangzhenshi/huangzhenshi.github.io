技能清单（智联招聘)：

  1. JAVA基础扎实，理解IO、集合等基础框架，
  2. 熟悉常用的Java开源框架（Spring、ibatis、struts等）；并对于使用过的开源框架，能了解到它的原理和机制；
  3. 熟悉分布式系统的设计和应用，熟悉分布式、缓存、消息等机制；能对分布式常用技术进行合理应用，解决问题；
  4. 掌握多线程及高性能的设计与编码及性能调优；有高并发应用开发经验；
  5. 掌握Linux 操作系统和大型数据库（Oracle、MySql）；对sql优化有一定的经验；
  6. 熟悉各种常用设计模式
  7. JVM原理
  8. 熟悉主流的key-value存储系统；

分解后按照技能相关度的计划

1） 集合、数据结构、算法 （半个月 ）（Done)
2） JVM 原理 java的内存结构 垃圾回收机制 监控工具（1个月）(Done)
3） linux （半个月）(Done)
4） 消息队列、缓存、分布式（1个月）
9） IO NIO （半个月）
5） Nosql（半个月）
6） oracle mysql 原理和调优，特别是mysql（1个月）
7） 多线程 和并发（1个月）
8） Spring mybatis stutus2 源码 和设计模式（2个月）
