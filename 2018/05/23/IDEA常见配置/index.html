IDEA的优点：

一次受累，3-5年以上的受益期，而且之前有一定的认知，找找笔记，上手更快
1）界面优美，全部纯黑，看起来爽，比Myeclipse 调过之后还爽。
2）持续更新，支持更高版本的JDK，不像Myeclipse 2014都不支持1.8
3）支持更多灵活的快捷键
4）自动补全功能
5）对html、js、css更好的支持（比如html元素有更多更准确的自动完成）
6）对新技术有更好的支持，gradle等新技术
7）支持配置文件导入的形式，可以迅速复制到其他Intellij上（File importSetting功能）

功能特性

参考博客
http://www.tiantianbianma.com/idea-top10-features-one.html/

  • Distraction Free Mode，类似于Ctrl + M

  • 代码助手是 Ctrl+Space(空格)，利用这个基本助手(Basic completion)可以完成变量、类型、方法、表达式等的最基本完成建议。而且，如果连续两次按下 Ctrl+Space，它会提供出更多的选择，包括私有成员和一些还未引入当前文件的静态成员等

开发功能配置

  1. 添加JDK环境和jar包

  • file > project Struture 里面设置JDK路径

  • 添加jar包 modules > 项目名 > dependence 添加即可

  1. 配置tomcat和热加载资源
    https://blog.csdn.net/maomengmeng/article/details/52043928/

  • 添加tomcat - server，配置server和 deployment

  • 设置project-setting 里面的module里面的web，设置对应的web.xml路径 和webapp路径

  • 设置 tomcatSetting > on fream deaction为 任意改变都触发，就是eclipse的build automatic

通用功能配置

  1. 配置文件导入功能
    file > import setting ，可以实现一次配置到处可用的效果。
