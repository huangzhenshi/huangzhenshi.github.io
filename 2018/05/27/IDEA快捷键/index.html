IDEA快捷键和高级设置

By Huang zhenshi

发表于 2018-05-27

  1. 1. 查询和设置简介
  2. 2. 查询类

查询和设置简介

  1. Setting 里面支持Kepmap 里面 根据action的名字查找相关的内容，也支持根据快捷键找action的名字
  2. 根据action查找快捷键的快捷键（find action） ，Ctrl + Shift + A(ction)

查询类

  1. 查找所有文件，Double Shift
  2. 查找所有类， Ctrl +Shift +T
  3. 查看最近打开过的文件列表， Ctrl + E

工具类

  1. 编辑器最大化（hide all tool windows），Ctrl + Q 还有双击都会触发 ,两次使用可以恢复，可以恢复神技+1

  2. Debug console最大化（Maximize Tool windows），Ctrl + UP，两次使用会恢复效果，用来查看console的输出

  3. 关闭一个打开的工具栏（Hide side tool windows），Ctrl + left，比如关闭console显示框

  4. 关闭断点显示框，(show gutter icons),Ctrl + 小键盘1 ，两次使用会恢复断点栏，这样可以获取代码显示栏的最大利用化

  5. 忽略断点运行至， Ctrl + Alt + F9

  6. 关闭文档，close file， Ctrl + W ，关闭全部 Ctrl + Shift + W

  7. 打开到物理位置（show in explorer), Ctrl + Shift + E

  8. 对齐、代码美化 （reformat code）（Ctrl + Shift + F )

  9. 代码注释和eclipse，区别是取消注释也是一样的“/” 而不是“\”

自动生成类

  1. 生成get set等 （generate constructor get set）， Ctrl + Insert

  2. 生成常用 if/ else if/ try/ catch finally/ while等模板代码， (surround with …)，Ctrl +小键盘2

  3. 智能自动补全，Ctrl + Shift + Enter，比如输入 if，else if， try都会自动补全
