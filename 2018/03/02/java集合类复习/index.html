继承关系

  • Collection接口下面：Set接口（无序不重复链表)、Queue接口（队列，先进先出)、List接口（有序数组)
  • Set实现类：HashSet，Queue实现类:linkedList，List实现类ArrayList。
  • Map接口下面：HashMap、HashTable

LinkedHashMap

  • 继承自HashMap，提供按照插入顺序获取元素的功能
  • 额外有一个header的双向链表维护元素插入的顺序，header本身为null，header.next为第一个元素，header.before为最后一个元素
  • 重写了Entry，额外的添加了before和after，保留了next

HashTable

  • 功能和HashMap类似，但是线程安全，put get size remove的时候都加了锁，而且是锁住了整个map，但是并发读取的时候就会慢等待,并行访问变串行访问。

ConcurrentHashMap

相对于Hashtable的改进：

  • 从锁整个map改进为锁一个segment
  • 从锁整个map到读valitle的值
  • extends AbstractMap implements ConcurrentMap

  • 并发原理分析：通过把数据分为16个段，每个段上再分一个HashMap，这样并发时就是锁住一个段，而不是整个map。而且只在put的时候锁一个segment，在get的时候是不锁的，获取的是valitle的值。

  • 两次与运算分别获取在segements段位置和hashmap数组中的下标位置，第一个经过了位移，确保两次的位置不同，这样两次结果就不同更分散

ArrayList和 Vector

Arraylist线程不安全，vector线程安全。为了提高效率和节约空间，最好指定大小初始化。
